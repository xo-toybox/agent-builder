# Agent Builder Roadmap

## v0.0.2 - Next Release

Features deferred from v0.0.1 for next iteration.

| Feature | Description |
|---------|-------------|
| Skills | Functional skills panel with skill creation and management |
| Agent config DB | Migrate agent config persistence from JSON file to database |
| Gmail Push | Gmail Watch, Pub/Sub for real-time email notifications |
| Async HITL | Queue schema, Slack notifications for background review |
| Deployment | Cloud Run/ECS, Postgres, Redis architecture |
| Memory schema | Priority rules, contact context, writing style persistence |
| Security | Token encryption, PII redaction, prompt injection mitigation |
| Success metrics | Triage accuracy, draft acceptance rate, time saved |
| E2E tests | Playwright end-to-end testing |
